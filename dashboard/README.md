# Belqees Media - Dashboard/CMS

لوحة تحكم مخصصة لإدارة محتوى موقع بلقيس ميديا، مبنية على React + TypeScript + Tailwind CSS.

## 🚀 البدء السريع

```bash
# تثبيت dependencies
npm install

# تشغيل في وضع التطوير
npm run dev

# بناء للإنتاج
npm run build

# معاينة البناء
npm run preview
```

## 📁 هيكل المشروع

```
src/
├── components/      # مكونات Dashboard
├── pages/           # صفحات Dashboard
├── layouts/         # التخطيطات
├── hooks/           # Custom Hooks
├── services/        # API Services
├── store/           # State Management
└── utils/           # Utilities
```

## 🛠️ التقنيات

- **React 18+** مع TypeScript
- **Tailwind CSS 3+** للتصميم
- **React Query** لإدارة البيانات
- **React Hook Form** للنماذج
- **Zod** للتحقق من البيانات
- **Tiptap** لمحرر النصوص الغني
- **Recharts** للإحصائيات
- **Vite** كأداة البناء
