# دليل إضافة صور الشركات التي وثقت بنا

## 📁 مكان رفع الصور

ضع صور شعارات الشركات في المجلد التالي:
```
public/images/
```

## 📝 أسماء الملفات المقترحة

يمكنك استخدام أسماء واضحة للصور، مثل:
- `partner-1.png` أو `partner-1.jpg`
- `partner-2.png` أو `partner-2.jpg`
- `partner-3.png` أو `partner-3.jpg`
- `partner-4.png` أو `partner-4.jpg`
- `partner-5.png` أو `partner-5.jpg`
- `partner-6.png` أو `partner-6.jpg`

**أو** يمكنك استخدام أسماء الشركات مباشرة (بالإنجليزية):
- `company-name-1.png`
- `company-name-2.png`
- إلخ...

## 📄 الملف الذي تحتاج تعديله

الملف: `components/pages/HomePageContent.tsx`

السطور: من 157 إلى 162

## 🔧 كيفية التعديل

بعد رفع الصور في `public/images/`، قم بتحديث المسارات في الملف:

### مثال:

إذا رفعت الصور بهذه الأسماء:
- `partner-1.png`
- `partner-2.png`
- `partner-3.png`
- `partner-4.png`
- `partner-5.png`
- `partner-6.png`

فستكون المسارات:
```tsx
{ id: '1', name: 'اسم الشركة 1', logo: '/images/partner-1.png' },
{ id: '2', name: 'اسم الشركة 2', logo: '/images/partner-2.png' },
{ id: '3', name: 'اسم الشركة 3', logo: '/images/partner-3.png' },
{ id: '4', name: 'اسم الشركة 4', logo: '/images/partner-4.png' },
{ id: '5', name: 'اسم الشركة 5', logo: '/images/partner-5.png' },
{ id: '6', name: 'اسم الشركة 6', logo: '/images/partner-6.png' },
```

## 📐 مواصفات الصور الموصى بها

- **الحجم**: 140x70 بكسل (أو نسبة 2:1)
- **الصيغة**: PNG (شفاف) أو JPG
- **الخلفية**: شفافة (PNG) أو بيضاء
- **الجودة**: عالية الوضوح

## 🔗 إضافة روابط للشركات (اختياري)

يمكنك أيضاً إضافة روابط لمواقع الشركات:

```tsx
{ id: '1', name: 'اسم الشركة 1', logo: '/images/partner-1.png', url: 'https://example.com' },
```

## 📋 ملخص الخطوات

1. ✅ ارفع صور الشعارات في `public/images/`
2. ✅ افتح ملف `components/pages/HomePageContent.tsx`
3. ✅ ابحث عن السطور 157-162
4. ✅ أضف المسارات الصحيحة للصور في حقل `logo`
5. ✅ غيّر أسماء الشركات في حقل `name` إذا أردت
6. ✅ احفظ الملف

---

**ملاحظة**: بعد رفع الصور، تأكد من أن المسارات تبدأ بـ `/images/` وليس `public/images/`
